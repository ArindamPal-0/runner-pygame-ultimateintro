## Setup

### Dev Setup (windows)

creating virtual environment and installing dependencies.

```powershell
$ mkdir .venv
$ pipenv install --dev
$ pipenv shell
```

dev setup from scratch.

```powershell
$ mkdir .venv
$ pipenv install
$ pipenv shell
$ pipenv install pygame
```

### Running the game

```powershell
$ python main.py
```
